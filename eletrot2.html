Circuito RLC série — análise em frequência fixa

—Circuito
⚡ Preencha os dados para ver a análise do circuito.
Triângulo de Impedância — R, X e Z em escala real
Diagrama Fasorial — tensões no circuito (I como referência)
Triângulo de Potências — P, Q e S
Reatâncias
\[X_L = 2\pi f L\]
\[X_C = \frac{1}{2\pi f C}\]
Impedância
\[Z = \sqrt{R^2 + (X_L - X_C)^2}\]
\[\varphi = \arctan\!\frac{X_L - X_C}{R}\]
Corrente e Tensões
\[I = \frac{V}{Z}\]
\(V_R = RI\) · \(V_L = X_L I\) · \(V_C = X_C I\)
Potências
\[P = RI^2\quad Q = XI^2\quad S = ZI^2\]
\(FP = \cos\varphi = P/S\)
Ressonância
\[f_0 = \frac{1}{2\pi\sqrt{LC}}\]
Em \(f_0\): \(Z_{min}=R\), \(I_{max}=V/R\), \(\varphi=0\)
Fator de Qualidade
\[Q = \frac{1}{R}\sqrt{\frac{L}{C}} = \frac{f_0}{BW}\]
\(BW = R/(2\pi L)\) Hz
